== Description ==
A very powerful plugin that will make Thesis Theme costomization more flexible. You are not required to know PHP to edit Thesis now. I have huge list of features that I will add one by one as the developement continues.

Features of the plugin

*   Flexibility in Editing the Header area of the theme.
*   Flexibility in Editing the Footer area of the theme.
*   Create widget areas with just click of your mouse (Above Header area, Below Header area, In the footer area. More to be added soon)
*   2 menus with flexibility to align left, right or center

== Installation ==

1. Upload `ultimate-thesis-options` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the Settings > Ultimate Thesis Options to change setting
